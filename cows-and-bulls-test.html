2 Player Duel

Both players enter a secret number and take turns guessing.

Both secrets stay hidden. Then the game begins with alternating turns.

Rules

  1. The number can never start with 0.
  2. You cannot repeat digits.
  3. Cows = number of digits correct overall.
  4. Bulls = number of digits in the correct place.
  5. Example: secret 51380, guess 15376 → 3 cows and 1 bull.
About online play

This version now supports simultaneous 2-player turn-taking on one shared device. For true remote online play with room codes, both players on separate devices would need a backend or realtime database like Firebase or Supabase.
